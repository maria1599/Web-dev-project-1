About the implemented features

Below are the list of features inplemented in all html pages using javascript and css.

  1. In the navigation bar of all html webpages, the link in the nav bar is highlighted when user is on that current page.
  2. ALl fields in Register.html and Order.html are required to be filled in, if not all fields or there are missing fields not filled in, an error message will show.
  3. In Register.html, if no is selected in "Would you like to tell us about your favourite pizza, the div with id="favpizza" will be hidden, and users can proceed to submit form without receiving error message from the fields inside that div.
  4. In Register.html, if Email and Re-type email are not the same, error message will be shown upon submit. This also applies to password and re-type password. In addtion, password and re-type password must be 8 characters long.
  5. In Order.html, Pick up is the default option in the intitial load of page in which automatically hides the billing address, and same as delivery checkbox for billing address. In addition, online is also the default option for 'payment method which shows the online method for credit card information.
  6. In Order.html, if users selects delivery method, the div for delivery address fields shows up as well as the same as delivery address. In addition if delivery method is selected, the payment method will be hidden. Thus, no error message will be received from the payment method if delivery method is selected.
  7. In Order.html, if 'same as delivery address is ticked after users filled in their delivery address, billing address will be automatically filled in. If 'same as delivery address is ticked before filling in the fields in delivery address, error message will be shown telling users to fill in delivery address first.
  8. In Order.html, postcode must be 4 digit long, and cvv/ credit card pin must be 3 digit long.
  9. In Order.html, if users select their credit card type, credit card number must be of a specific numbered of digits entered. Such as , if users select visa or master card, the credit card number must be 16 digits long, and if users select american express, their credit card number must be 15 digits long.
